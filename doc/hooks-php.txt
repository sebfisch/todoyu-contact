Contact Extension - PHP Hooks
=============================



1.	Event Hooks		Evoked via TodoyuHookManager::callHook($ext, $name, Array $params)
---------------
	renderPersonDetailView
	@param	Integer	idPerson
	@return	String
	Evoked before person-detail view is rendered. Content is attached at the end

	renderCompanyDetailView
	@param	Integer	idCompany
	@return	String
	Evoked before company-detail view is rendered. Content is attached at the end


2. Event Data Modification
--------------------------
	personCSVExportParseData
	@param	Array			$exportData
	@param	TodoyuContactPerson	$person
	@return	Array
	Evoked after person data is parsed for the export. Its possible to modify the exportData array. Has to give it back.

	companyCSVExportParseData
	@param	Array			$exportData
	@param	TodoyuContactCompany	$person
	@return	Array
	Evoked after company data is parsed for the export. Its possible to modify the exportData array. Has to give it back.



3. Form Hooks (Build Form, Load Data, Save Data)
------------------------------------------------
	@see	all forms in ext/contact/config/form
